# E-Commerce-Business-Performance-Marketing-Roi-Analysis

## Project Overview
This project analyzes a multi-year e-commerce dataset to identify key growth drivers, regional performance, and marketing effectiveness.
I used Python (Pandas) to transform raw data into actionable business insights.

## Key Insights
- **Top Performer:** Home Decor emerged as the leading category in both revenue ($5M+) and customer satisfaction (3.13 rating).
- **Profitability:** Groceries maintained the highest profit margin at 21.1%.
- **Marketing ROI:** Discovered a very weak correlation (0.015) between Ad Spend and Revenue, suggesting a need for a new marketing strategy.
- **Regional Strength:** The North region is the primary revenue hub ($7.6M+).

## Tech Stack
- Python (Pandas, NumPy)
- Jupyter Notebook
- Data Analysis (Correlation, GroupBy, Aggregation).
